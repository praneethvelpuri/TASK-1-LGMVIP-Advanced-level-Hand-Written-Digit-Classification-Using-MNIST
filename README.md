# Hand-Written Digit Recognition

This project is a simple example of how to recognize hand-written digits using machine learning.

## Requirements

* Python 3
* NumPy
* Scikit-learn
* Matplotlib
  
## Results

The model achieves an accuracy of about 98% on the MNIST dataset.
